# Mini Project 1: Student Performance Analyzer

## Question: Write a Python program to calculate a student's academic performance.

### Requirements
Ask the student to enter marks for 5 subjects.
- Calculate:
- Total Marks
- Average Marks
- Percentage
- Display all the calculated values.

### Constraints
Use variables, input(), print(), arithmetic operators, and type casting only.
Do not use loops or conditional statements.

# Mini Project 2: Fitness & BMI Calculator

## Question: Develop a Python program to calculate a person's BMI (Body Mass Index).

### Requirements
- Ask the user to enter:
- Name
- Age
- Height (in meters)
- Weight (in kilograms)
- Calculate BMI using:

**BMI = Weight / (Height × Height)**

### Display:
- User Details
- BMI Value
- Constraints

*Use variables and arithmetic operators only.*
